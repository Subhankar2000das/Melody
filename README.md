# 🎵 Melody – Full Stack Music Streaming Platform

Melody is a modern, full-stack music streaming web application built with **Next.js 16, Supabase, and TanStack Query**.  
It provides a seamless experience for users to explore, play, and organize music while giving admins powerful tools to manage content and users.

## ✨ Key Features

### 🎧 User Experience
- 🎵 Play songs with a custom music player
- ❤️ Like / Unlike songs
- 📚 Create and manage playlists
- 🔁 Recently played tracking
- 🔍 Smart search with live suggestions (onChange)
- 🎼 Browse albums and songs
- 🎨 Spotify-style "Liked Songs" UI
- 📱 Fully responsive design

---
